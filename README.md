# Where's Waldo

## General Info

Interactive game of Where's Waldo. Find Waldo and click on him. The faster you are, the bigger your chance of getting into the top 10!

[Live Preview](#)

## Technologies

- HTML, CSS, Javascript
- React
- Styled Components
- Firebase
- Jest, React Testing Library
